Que son los componentes pasivos en la red?

Se refieren a los dispositivos o elementos que no requieren energía activa para su funcionamiento y que no amplifican o alteran las señales eléctricas que transmiten.

Fin

Que son los componentes activos en la red?

Los componentes activos en una red de computadoras son dispositivos que requieren energía para funcionar y que tienen la capacidad de amplificar, procesar o modificar las señales de datosmplifican o alteran las señales eléctricas que transmiten.

Fin

componentes activos

Swicth

Un switch es un dispositivo de red que conecta varios dispositivos dentro de una misma red local (LAN).

Router

Un router es un dispositivo de red que se utiliza para conectar diferentes redes entre sí, como una red local (LAN) con una red externa (por ejemplo, Internet).

Tarjeta Red

Una tarjeta de red es un componente de hardware que permite que un dispositivo (como una computadora o un servidor) se conecte a una red.

Interruptor inteligente WIFI

Un interruptor inteligente Wi-Fi es un dispositivo que permite controlar de manera remota el encendido y apagado de aparatos eléctricos o luces a través de una conexión Wi-Fi y una aplicación móvil o un asistente de voz.

cableado estructurado

Ponchadora: es una herramienta manual utilizada para insertar cables en conectores de tipo IDC
Crimpodora: es una herramienta utilizada para fijar conectores modulares (como RJ45, RJ11, RJ12)
Desforradora:es una herramienta utilizada para remover el aislamiento exterior de los cables sin dañar los conductores internos.
Tester:=es una herramienta que se utiliza para verificar la integridad y correcta terminación de un cable de red.
